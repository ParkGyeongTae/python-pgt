# python-pgt

## conda 연결
### source /Users/pgt0409/opt/anaconda3/bin/activate

## 리스트 조회
### conda info --envs
### conda env list

## 가상환경 활성화
### conda activate <가상환경 이름>

## 가상환경 비활성화
### conda deactivate

## 가상환경 제거
### conda env remove -n <가상환경 이름>

## 가상환경 만들기
### conda create -n <가상환경 이름> python==<파이썬 버전> -y

## 라이브러리 설치
### conda install -c conda-forge pyspark -y
